framewise · widget specification · v0.1

A widget set that explains itself.

Sharp corners, hairline borders, monospaced numerics. One accent — rust — reserved for focus, drag, and primary action. Every widget describes its state explicitly; nothing is hidden behind animation or chrome.

Scrollbars

always visible. thumb length encodes how much of the content fits in view; thumb position encodes scroll offset. dragging shifts the thumb to rust.

fn frame(ctx: &mut Ctx) {
  ctx.window("Inspector", |w| {
    w.label("position");
    w.drag("x", &mut pos.x);
    w.drag("y", &mut pos.y);
    w.separator();
    w.label("size");
    w.drag("w", &mut size.w);
    w.drag("h", &mut size.h);
    w.separator();
    w.label("frame");
    w.text(&format!("{} fps", fps));
    w.slider("alpha", &mut a, 0.0..1.0);
  });
}
